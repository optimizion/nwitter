# Nwitter - Firebase를 활용한 Twitter Clone 앱

- Firebase Authentication을 통한 깃허브, 구글 회원가입 및 로그인
- Firebase Firestore, Storage를 통한 유저 트윗, 이미지 저장
- React Router를 사용한 SPA 구현
